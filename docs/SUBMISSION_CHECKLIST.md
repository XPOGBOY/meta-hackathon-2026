## If Something Breaks

| Problem | Fix |
|---|---|
| Tests fail | Run `python -m pytest tests/ -v --tb=long` and fix the specific test |
| Server won't start | Check port 7860 is free: `netstat -an | findstr 7860` |
| Multi-robot import error | Check `warehouse_env/server/multi_robot_environment.py` exists |
| Inference crashes | Run `python inference.py 2>&1 | head -40` to see first error |
| HF Spaces down | Have local server + video as backup |
